Icon Attribution
================

The default icons included in ComfiHierarchy are from:
- Icons8.com (https://icons8.com)
- VRChat community assets (where noted in subfolder names)

These icons are used with appropriate licensing.

For custom icons:
- Place your own PNG files in the Icons/Custom/ folder
- Name them according to the component type (e.g., MyComponent.png)
- Icons will be automatically detected and loaded
